# 🌐 IP Subnet Calculator

A fully client-side IPv4 subnet calculator built with pure HTML, CSS, and JavaScript — no frameworks, no backend, no dependencies. 
It supports both dotted notation (`255.255.255.0`) and CIDR (`/24`) input, includes a quick CIDR prefix picker, binary bit visualization, IP classification (Private/Public/Loopback/Multicast), and a subnet division table showing the first 8 subnets. Just open `index.html` in any browser, or host it instantly via **[GitHub Pages](https://<your-username>.github.io/subnet-calculator/)** by setting the source to `main` branch in **Settings → Pages**.
