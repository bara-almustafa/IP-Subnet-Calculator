# 🌐 IP Subnet Calculator

A fully client-side IPv4 subnet calculator built with pure HTML, CSS, and JavaScript —
no frameworks, no backend, no dependencies.

## ✨ Features

- 🧮 **Full subnet calculation** — network address, broadcast, first/last host, wildcard mask, usable & total hosts
- 🔢 **Dual input support** — dotted notation (`255.255.255.0`) and CIDR (`/24`) format
- ⚡ **Quick CIDR picker** — clickable `/8` – `/32` badge buttons
- 🟦 **Binary visualization** — color-coded bits separating network vs host portions
- 🏷️ **IP type detection** — Private (RFC1918), Public, Loopback, Link-Local, Multicast, Reserved
- 📊 **Subnet division table** — first 8 subnets with full host ranges
- ✅ **Input validation** — real-time error highlighting on invalid input
- ⌨️ **Keyboard friendly** — press `Enter` to calculate
